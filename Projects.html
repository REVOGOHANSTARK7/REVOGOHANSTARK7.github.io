PROJECT OVERVIEW

  • Farmers connect is cross platform mobile application built using react native as front end and firebase as back end database for storage. This project focuses on solving many farmer issues that they are facing in the current situations like knowing crop information, pesticides usage, fertilizers usage and etc. The team of 6 members are involved to create this app. This app is developed by following scrum methodology. 8 Sprints, each sprint is of week are planned to complete this app including training.

TECHNOLOGIES USED

  • FRONTEND : React Native, Expo, Android Emulator
  • UI/UX: Figma, BuilderX
  • BACKEND : Firebase, Code Versioning Systems: Git and GitHub, Google developer console
